# 🚀 Ethereum Wallet using Web3.js & MetaMask

## 🎯 Project Overview
This is a simple Web3 Ethereum Wallet built with **Web3.js** and **MetaMask**. It allows users to:

✅ Connect MetaMask Wallet  
✅ Display Wallet Address  
✅ Check ETH Balance  
✅ Send ETH to another address  

---

## 🌟 Tech Stack
- HTML5  
- CSS3  
- JavaScript  
- Web3.js  
- MetaMask
